Este projeto é o backend de um sistema de controle de ativo imobilizado, desenvolvido em Node.js com Express. A aplicação permite o gerenciamento de ativos de uma organização, incluindo cadastro, atualização, consulta e exclusão de informações patrimoniais. Utiliza MongoDB para armazenamento dos dados, JWT para autenticação segura, Bcrypt para proteção de senhas e Nodemailer para envio de notificações por e-mail. O sistema foi pensado para garantir a segurança, escalabilidade e fácil integração com frontends diversos.
